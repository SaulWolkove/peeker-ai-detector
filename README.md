# Peeker - AI Image and Text Detector

A Chrome extension that detects AI-generated images and text using machine learning models running entirely in your browser.

## Features

- **AI Image Detection** - Analyze images to determine if they were generated by AI
- **AI Text Detection** - Detect AI-generated text content
- **Drag & Drop** - Simply drag images onto the floating icon to analyze
- **Right-Click Menu** - Right-click any image and select "Run Peeker AI Detection"
- **Privacy-Focused** - All analysis runs locally in your browser using [Transformers.js](https://huggingface.co/docs/transformers.js)

## Installation

### From Source

1. Clone the repository:

   ```bash
   git clone https://github.com/SaulWolkove/peeker-ai-detector.git
   cd peeker-ai-detector
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Build the extension:

   ```bash
   npm run build
   ```

4. Load in Chrome:
   - Open `chrome://extensions/`
   - Enable "Developer mode" (top right)
   - Click "Load unpacked"
   - Select the project folder

## Usage

### Analyze Images

**Option 1: Drag & Drop**

- Drag any image onto the Peeker floating icon that appears on web pages

**Option 2: Right-Click**

- Right-click on any image
- Select "Run Peeker AI Detection" from the context menu

### Analyze Text

- Select text on any webpage
- Use the Peeker interface to analyze the selected text

## Development

```bash
# Install dependencies
npm install

# Build once
npm run build

# Watch mode (rebuilds on changes)
npm run dev
```

## Tech Stack

- **Chrome Extension Manifest V3**
- **Transformers.js** - Run ML models in the browser
- **Webpack** - Module bundling
